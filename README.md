# PDFViewer in SwiftUI

A PDF Viewer built in SwiftUI and PSPDFKit for iOS 14 and macOS Big Sur.

Requires Xcode 12 GM for iOS 14, recommended Xcode 12.2b2 or higher for macOS Big Sur.
